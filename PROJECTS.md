### 📦 Developer Tooling & Open Source

#### [Express Forge (express-cli)](https://github.com/CODE-Y02/express-cli) · [Docs](https://code-y02.github.io/express-cli/)
*CLI for scaffolding production-ready Express + TypeScript backends.*
- Go from zero to a fully structured API in under 30 seconds — pre-integrated Auth, ORM, Docker, and OpenAPI
- **Stack**: TypeScript, Node.js, Prisma, Sequelize, Docker, Zod, Vitest

#### [@express-route-cache](https://github.com/CODE-Y02/express-route-cache) · [npm](https://www.npmjs.com/package/@express-route-cache/core)
*TypeScript-first TTL and in-memory route caching middleware for Express.js.*
- ~100 daily downloads · pluggable cache adapters · configurable per-route TTL · cache invalidation
- Zero runtime dependencies beyond Express · fully typed with TypeScript generics
- **Stack**: TypeScript, Node.js, Express.js
